Vagrant

What is it?

“Vagrant provides easy to configure, reproducible, and portable work environments built on top of industry-standard technology and controlled by a single consistent workflow to help maximize the productivity and flexibility of you and your team.”

Seriously though

Vagrant makes it easy to create, configure, provision, and share virtual machines.

Vagrant machines are:

  • Portable
  • Self-contained
  • Lightweight
  • Reproducible

Portable

  • Runs on Windows / OS X / Linux
  • Defaults to free VirtualBox software
  • Self-contained installer

Self-contained

  • Only contains what you want it to contain
  • Does not require developer tools on the host machine
  • Does not install any libraries / servers on the host machine

Lightweight

  • Reduces virtual machine state (large binaries) to code (small text files)
  • Maps a hard-to-manage domain into one that is easily managed with Version Control
  • Base images can be shared among developers and among projects

Reproducible

  • Starts with known (shareable) base image
  • Applies configuration and customization
  • Eliminates "works on my machine" problems
